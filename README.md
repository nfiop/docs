# nfiop - NAND flash I/O platform

A platform for NAND flash I/O - simulation, emulation, data extraction
and a sandbox for storage experiments.

There's a [whole document about this](Purpose.md).

## What is the expected development flow?

In the purpose document I explain a lot about the motivation and what
I want to do in this project.

To summarize, there's this [document about development stages](Development.md).

## Do you accept patches? who funds this?

This is a project I do because of pure curiosity around this topic. As
such, nobody funds this project, and I consider this as a research project
to some extent.

I intend to learn and implement what is possible, so this is by no means
a production-made or production-ready product in any shape or form.

I will probably not accept patches, as I prefer for now to build the
entire thing alone. Suggestions are welcome as always.
